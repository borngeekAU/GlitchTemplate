================================================================================
                    AUDIO ELEMENTS DIRECTORY
                    RUKSPIN ARCHIVE V2.1
================================================================================

This directory is designated for storing audio assets.

RECOMMENDED FILE STRUCTURE:
--------------------------------------------------------------------------------

/audio_elements
├── /music                  # Background music tracks
│   ├── ambient_001.wav
│   ├── glitch_beat_001.wav
│   └── ...
│
├── /sfx                    # Sound effects
│   ├── static_noise_001.wav
│   ├── signal_beep_001.wav
│   ├── vhs_tracking_001.wav
│   ├── glitch_hit_001.wav
│   └── ...
│
├── /voice                  # Voice-over elements (if any)
│   └── ...
│
├── /loops                  # Loopable audio elements
│   ├── static_loop_001.wav
│   └── ...
│
└── /masters               # Final mixed audio
    └── ...


NAMING CONVENTION:
--------------------------------------------------------------------------------

Format: {CATEGORY}_{DESCRIPTION}_{NUMBER}.{EXT}

Examples:
- static_noise_001.wav
- glitch_transition_001.wav
- ambient_drone_001.wav
- signal_detected_001.wav


RECOMMENDED SPECIFICATIONS:
--------------------------------------------------------------------------------

MASTER FILES:
- Format: WAV (uncompressed)
- Sample Rate: 48kHz
- Bit Depth: 24-bit
- Channels: Stereo

EXPORT FILES:
- Format: MP3 or AAC
- Bitrate: 320kbps (MP3) or 256kbps (AAC)
- Sample Rate: 48kHz


SUGGESTED SOUND ELEMENTS FOR GLITCH_STITCH:
--------------------------------------------------------------------------------

ESSENTIAL:
[ ] Static/white noise (various intensities)
[ ] VHS tracking sound

[ ] Glitch/data corruption sounds

TRANSITIONS:
[ ] Glitch hit/impact
