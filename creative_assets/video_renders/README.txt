================================================================================
                    VIDEO RENDERS DIRECTORY
                    RUKSPIN ARCHIVE V2.1
================================================================================

This directory is designated for storing video content files.

RECOMMENDED FILE STRUCTURE:
--------------------------------------------------------------------------------

/video_renders
├── /transmissions          # Main video content series
│   ├── TRANSMISSION_001.mp4
│   ├── TRANSMISSION_002.mp4
│   └── ...
│
├── /shorts                 # Platform-optimized versions
│   ├── /tiktok            # 9:16, optimized for TikTok
│   ├── /reels             # 9:16, optimized for Instagram
│   └── /youtube_shorts    # 9:16, optimized for YouTube
│
├── /raw                    # Unedited source footage
│   └── ...
│
├── /exports               # Final exports ready for upload
│   └── ...
│
└── /archive               # Older/retired content
    └── ...


NAMING CONVENTION:
--------------------------------------------------------------------------------

Format: {TYPE}_{NUMBER}_{VERSION}_{PLATFORM}.{EXT}

Examples:
- TRANSMISSION_001_v1_master.mp4
- TRANSMISSION_001_v1_tiktok.mp4
- TRANSMISSION_001_v1_reels.mp4
- STATIC_FRAME_001_v1.png


RECOMMENDED SPECIFICATIONS:
--------------------------------------------------------------------------------

MASTER FILE:
- Resolution: 1080x1920 (9:16) or 1920x1080 (16:9)
- Codec: H.264 or H.265
- Bitrate: 10-20 Mbps
- Audio: AAC, 48kHz, Stereo

TIKTOK EXPORT:
- Resolution: 1080x1920
- Duration: 15-60 seconds
- Max size: 287.6 MB

INSTAGRAM REELS:
- Resolution: 1080x1920
- Duration: 15-90 seconds
- Max size: 4 GB

YOUTUBE SHORTS:
- Resolution: 1080x1920
- Duration: Up to 60 seconds
- Format: MP4, MOV
